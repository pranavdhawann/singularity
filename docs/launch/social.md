Singularity `v0.2.0` is an early local-first memory and permission layer for AI assistants, built for technical early adopters.

Import ChatGPT history and project files. Retrieve cited, source-backed context. Run fully offline, with Ollama, or with an OpenAI-compatible provider. Before any external call, inspect and approve the exact locally redacted prompt.

Try the one-command Docker demo or a local pnpm install with no cloud account. Desktop packaging is not included yet, and the SQLite timeline is not encrypted at rest.

https://github.com/pranavdhawann/singularity
